## 🌟 Overview

**HealthScore AI** is a comprehensive health assessment platform that helps users understand their body health through 6 different AI-powered tests. Get personalized nutrition recommendations, track your health monthly, and make informed wellness decisions.

## 🎯 Features

### Health Assessment Tests

1. **Body Fat Analyzer** - Calculate metabolic age and body composition
2. **Longevity Score** - Estimate life expectancy and biological age
3. **Heart Risk Checker** - Assess cardiovascular health risks
4. **Stress Level Analyzer** - Evaluate mental stress and wellness
5. **Mental Age Test** - Determine cognitive age and brain health
6. **Sleep Quality Score** - Analyze sleep patterns and quality

### Daily Health Tools

1. **Calorie Calculator** - Estimate daily caloric needs
2. **Water Intake Tracker** - Calculate optimal hydration
3. **Step Goal Calculator** - Determine ideal daily steps

### Monetization

- **Free Preview** - Basic test results for all users
- **Paid Reports** - Unlock detailed reports (one-time payment)
- **Monthly Subscription** - ₹199/month for continuous tracking
- **PDF Downloads** - Professional health reports
- **Nutrition Plans** - Personalized meal recommendations

- ✅ Smooth animations and transitions
- ✅ Interactive data visualizations
- ✅ WhatsApp-optimized sharing
- ✅ AI-powered chatbot support
- ✅ Multi-page navigation
- ✅ Subscriber dashboard

## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The app will be available at `http://localhost:5173`

## 🌐 Deployment

### Option 1: Vercel (Recommended)


### Option 2: Netlify


### Option 3: Custom Server

```bash

# Upload /dist folder to your server
```
